    Ejemplo de Componente de Cabecera

    Este es un ejemplo de cómo usar el componente de cabecera reutilizable en todas tus páginas.

    El encabezado incluye:

    • Logo con enlace a la página de inicio
    • Menú de navegación
    • Botón de hamburguesa para móviles
    • Animación de onda en la parte inferior
    • Menú desplegable para móviles con animación

    Cómo Implementarlo

    Para usar este componente en cualquier página:

    1. Incluir el script del componente
    2. Agregar el tag <header-component> en tu HTML
    3. Personalizar los enlaces del menú usando slots
