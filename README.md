# Digital Agency MVP - Promotional Website

A modern, responsive promotional website built to validate the MVP (Minimum Viable Product) for a digital agency offering web development, AI agents, and coding consulting services.

## 🎯 Project Overview

This project serves as a **proof of concept** and **MVP validation** for a digital agency's service offerings. The goal is to quickly test market demand and gather client feedback before investing in a full-scale Next.js application.

### Current Status: MVP Validation Phase
- ✅ **Static HTML/CSS Implementation** - Quick deployment for testing
- 🔄 **Market Validation** - Gathering client interest and feedback
- 🚀 **Next.js Migration Planned** - For production-ready SEO and performance

## 🏗️ Architecture & Technology Stack

### Current Implementation (MVP)
- **Frontend**: Vanilla HTML5, CSS3, JavaScript
- **Styling**: Custom CSS with CSS Grid and Flexbox
- **Animations**: CSS animations and JavaScript scroll effects
- **Deployment**: Static hosting (GitHub Pages, Netlify, etc.)
- **Forms**: Firebase Firestore integration for lead capture

### Planned Production Stack
- **Framework**: Next.js 14+ (App Router)
- **Styling**: Tailwind CSS or Styled Components
- **CMS**: Contentful or Sanity for content management
- **Analytics**: Google Analytics 4 + Hotjar
- **SEO**: Next.js built-in SEO optimization
- **Performance**: Vercel deployment with edge functions

## 🎨 Features

### Current MVP Features
- **Multi-page Structure**: Three specialized landing pages
  - `index.html` - Web Development Services
  - `asesorias.html` - Programming Mentorship & Consulting
  - `agentes.html` - AI Virtual Agents & Automation
- **Responsive Design**: Mobile-first approach with breakpoints
- **Interactive Elements**:
  - Animated header waves
  - Morphing WhatsApp button
  - Parallax background effects
  - Smooth scroll animations
- **Lead Generation**: Contact forms with Firebase integration
- **Dark/Light Mode**: Theme toggle with localStorage persistence
- **Accessibility**: ARIA labels and keyboard navigation

### Planned Production Features
- **Static Site Generation (SSG)** for optimal SEO
- **Server-Side Rendering (SSR)** for dynamic content
- **Image Optimization** with Next.js Image component
- **Internationalization (i18n)** for multi-language support
- **Advanced Analytics** and conversion tracking
- **A/B Testing** capabilities
- **Performance Monitoring** with Core Web Vitals
