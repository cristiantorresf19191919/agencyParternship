Header Include Example

This page demonstrates how to include the reusable header using JavaScript injection. The header is defined in a separate file and loaded onto this page.

Key benefits:

  • Header is maintained in a single file (js/header.js)
  • Updates to the header are applied across all pages
  • Maintains all original functionality including the hamburger menu and wave animation

How To Implement

To use this header on any page:

  1. Add a <div id="header-container"></div> where you want the header to appear
  2. Add <script src="js/header.js"></script> before the closing body tag
  3. Make sure the page includes the required CSS styles
